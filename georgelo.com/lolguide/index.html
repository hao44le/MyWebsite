LOL Guide

LOL Guide is an iOS app that provides various information of the most popular online game now, League of Legends. The app is available for download in iOS App Store, Windows Phone Store, and Windows Store.

I began working on the iOS version back in 2013. Several months after, my friend James Tsai ported the iOS Objective-C code to equivalent C# codes, which branched out to be Windows Phone and Windows versions.

  • Type - Personal / Group Project
  • Platform - iOS (iPhone), Windows Phone, Windows
  • Skills/Tools Used - Objective-C, Regex, C#, Core Data, SQLite, Xcode, Visual Studio
