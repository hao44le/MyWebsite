iMoTV

iMoTV is an app that allows the user to watch Taiwanese drama. Usually, the user has to go to various websites to find the specific drama and episode he or she wants to watch. But with this app, everything is organized.

I worked with my partner James Tsai on this app. This Windows Phone app basically retrieves content by crawling and parsing information from various websites.

My responsibility was to develop and maintain the parser, which uses regular expressions.

  • Type - Group Project
  • Platform - Windows Phone
  • Skills/Tools Used - C#, XAML, Regex, Visual Studio
